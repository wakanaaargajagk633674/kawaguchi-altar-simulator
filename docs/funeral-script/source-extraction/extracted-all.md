### 抽出内容
**＜Case1 〜絆〜＞**「間もなく開式でございます。本日は、{{deceasedName}}様のお人柄を偲び、遺されたご遺族様を想い、お別れを惜しむ多くの方がご参列して下さっております。{{deceasedName}}様が生前繋いでこられた人と人との絆は、今日こうしてゆかりのある方々を繋ぐ大切な時間を作って下さいました。」
**＜Case2 〜ご遺影〜＞**「間もなく開式でございます。開式までのひと時は、祭壇のご遺影を見つめて頂きながら、{{deceasedName}}様への感謝のお言葉や、伝える事ができなかったお言葉を、お心の中でお伝えいただければと思います。あのお写真は・・・」
**＜Case3 〜生年月日〜＞**「間もなく開式でございます。故 {{deceasedName}}。{{ceremonyDate}}、{{address}}の地に御出生。以来、激動の昭和、そして平成から令和へと移りゆく時代の中、…皆様と共に歩み続けてこられた尊いご生涯でございました。」
**【明るくて前向きな方】**「愚痴や弱音を一切吐かず、人の悪口を言わず、いつも前向きで朗らか。…『大事なのは、今この瞬間なんだ。』過去にとらわれず、未来を恐れず、今その時を精一杯生きてこられたご生涯。」
### 使えそうなフレーズ
- 開式前ナレーション（導入）：絆／ご遺影／生年月日の3型
- 人柄紹介：「愚痴や弱音を一切吐かず、人の悪口を言わず、いつも前向きで朗らか」
- 故人の言葉引用：「大事なのは、今この瞬間なんだ。」
### 注意点
故人の言葉『　』の前後は間を取る指示あり。導入はナレーション全体の起点。

# B. 司会マニュアル（作成法・無宗教式台本・ショート50編）

## 資料 No.039〜No.040
### 元画像
public/images/shikai/司会マニュアル/SKM_C251i24121614220_0001.jpg 〜 _0002.jpg
### 推定資料種別
司会台本／ナレーション作成マニュアル（開式前ナレーション）
### 読み取り状態
一部不明（本文が薄い。空欄プレースホルダ多数）
### 抽出内容
**【開式前 ナレーション】** 会場の私語を鎮め、参列者の気持ちを式に向ける役割。**《導入》** 開式時刻が近づいたら導入を入れ、故人を偲ぶ雰囲気を作る。導入例①ご逝去のご案内／②生年月日を入れる／③葬儀告別式について。**《本文》** ①故人略歴 ②疾病。**《結び》** 導入と同じ時代を振り返り締めコメント。
- 趣旨例「皆様、本日はお忙しい中、{{deceasedName}} 儀の通夜（葬儀・告別式）にご参列いただき、誠にありがとうございます。」「{{deceasedName}} 儀は、{{ceremonyDate}} 永眠いたしました。{{age}}歳でございました。」
