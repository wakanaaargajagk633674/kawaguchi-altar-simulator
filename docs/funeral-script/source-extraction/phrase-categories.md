# フレーズ分類（カテゴリ別整理）

> `extracted-all.md` の内容をカテゴリへ再整理。各項目に**代表フレーズ**と**出典 No.**を付す。
> 変数は `{{deceasedName}}` 等。地名/店名/日付/年齢は取材値に差し替える前提。
> 「F=固定テンプレート向き / AI=AI生成向き」のタグは `generation-policy.md` と対応。

## 式種別

### 通夜（仏式）　[F中心]
- 司会台本一式：誘導 → 前説 → ナレーション → 導師入場 → 開式の辞 → 焼香（喪主→遺族親族→弔問者）→ 焼香終了案内 → 導師退席 → 閉式（翌日葬儀案内）。出典 No.059–060
- 閉式は「閉式」ではなく**「区切り」**と表現。出典 No.020, 060

### 仏式 葬儀・告別式　[F中心]
- 司会台本一式：誘導 → 前説 → ナレーション → 導師入場 → 開式の辞（＋初七日併修版あり）→ 弔辞 → 弔電拝読 → 焼香 → 導師退席 → 閉式 → お別れの儀準備案内 → 移動。出典 No.061–062
- 出棺・火葬場同行の案内は前説に内包。出典 No.058

### 無宗教 告別式（お別れの会）　[F＋AI]
- 基本進行11ステップ：開式ナレ→開式の辞→黙とう→弔辞→弔電→故人略歴→思い出映像→お別れの言葉→献奏→献花→閉式の辞。出典 No.044–046
- 献奏・献花・閉式の辞は完成度の高い台本あり。出典 No.046

## 導師入場案内　[F]
- 「導師ご入場でございます。ご一同様、合掌をもってお迎え下さいませ。」→「お直り下さいませ。」出典 No.059, 061
- 作法は導師打合せで確認（起立合掌／着座合掌／お辞儀）。出典 No.055, 057

## 焼香案内　[F]
- 遺族親族「これより、ご遺族ご親族の皆様にご焼香へとお進み頂きます。喪主{{chiefMournerName}}様よりお進み下さいませ。」出典 No.059, 062
- 参列者「これより係りの者がご案内させて頂きます。順次ご焼香へとお進みくださいませ。」出典 No.059, 062
- 多数時の一回焼香依頼「ご焼香はお心を込めて一回焼香にて、合掌礼拝をお願い申し上げます。」出典 No.059, 062
- 焼香終了案内「いまだご焼香がお済みでない方は、ご焼香へとお進みくださいませ。」出典 No.060

## 献花案内　[F]（無宗教式）
- 「皆様それぞれの想いを一輪のお花に託し、ご献花へとお進み頂きます。喪主{{chiefMournerName}}様よりお進み下さいませ。」出典 No.046
- 終了後「献花台には、皆様お一人お一人から贈られた白いカーネーション…」出典 No.046

## 献灯案内　[F]

## 導師退場案内　[F]

- 起立後も作法が続く事があるので慌てない。出典 No.060

## 弔電紹介　[F]
- 拝読定型「多方面より弔電が寄せられております。これより順序不同にて拝読させて頂きます。」→本文2〜3通→「ご尊名のみを拝読させて頂きます。」→締め。出典 No.062
- 敬称対応表／読み仮名／弔電例文6種。出典 No.063

## 忌み言葉・避けた方がよい表現　[—]
- 抽出資料に明示の忌み言葉リストは**無し**。一般的な重ね言葉（重ね重ね・たびたび・再三）、「死ぬ/急死」直接表現、「迷う/浮かばれない」等は**要追補**（業界一般知識）。
