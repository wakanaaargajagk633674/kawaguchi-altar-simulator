## 共通：分岐させるべき条件（台本生成の入力フラグ）
- 焼香方法：立礼 / 回し焼香、一回焼香依頼の有無（参列者多数時）
- 初七日：繰上げ併修の有無（式中2回焼香）
- 弔辞 / 弔電 / 喪主挨拶 / 思い出映像 / 献奏 / 献花 / 献灯：各有無
- 出棺後：式場に戻る / 戻らない（手荷物移動案内の分岐）
- 火葬場同行・精進落とし同席の有無
- ナレーション：入れる / 入れない（葬儀の状況で省略あり。出典 No.053）
- 宗派：導師入退場の作法（起立合掌/着座合掌/お辞儀＝浄土真宗に多い。出典 No.057）
